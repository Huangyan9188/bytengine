Bytengine documentation
=======================

Bytengine is an HTTP based content repository API. It enables you to store JSON
and Binary Data in a pseudo hierarchical file system of files and folders.
Typical usage:

    * Content Management System backend
    * Document Management System backend
    * Image storage and delivery server
